## Beschrijving
Cool Kunst en Cultuur wil graag voor het prelude fonds een website om bij te houden waar de instrumenten zijn. op deze website kun je zien welke instrumenten er zijn, of ze zijn uitgeleend en of ze schade hebben. deze [link](https://server-side-rendering-server-side-website-mish.onrender.com) gaat naar de website die ik hier voor heb gemaakt. in deze eerste versie staat een home pagina met extra informatie over het prelude fonds, en een overzicht met alle instrumenten.
